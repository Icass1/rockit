# Welcome to Expo 👋

Created with `create-expo-app`.

> Building APK/IPA: See BUILDINGAPKIPA.md

## Start

1. Install: `npm install`
2. Run: `npx expo start`

Output shows options: dev build, Android emulator, iOS simulator, Expo Go.

Edit files in **app** directory. Uses file-based routing.

## Reset

```bash
npm run reset-project
```

Moves starter to app-example/, creates blank app/.
